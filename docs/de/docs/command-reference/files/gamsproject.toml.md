# gamsproject.toml

Diese Datei enthält die Projektkonfiguration für ein Gams-Projekt.
Es ist empfehlenswert, diese automatisch via

```bash
preprocess project init 
```

in einem Schritt mit einer basalen Projektstruktur anzulegen.

Details dazu finden sich in der Beschreibung des 
[Unterbefehls](../subcommands/project.md)
