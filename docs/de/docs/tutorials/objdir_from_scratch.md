# Objektverzeichnisse für neue Projektdaten

Liegt ein neues Projekt an, können die Objektverzeichnisse direkt erzeugt werden.

## Händische Erzeugung der Projektverzeichnisse

Das bedeutet, dass ein:e Bearbeiter:in das Verzeichnis per Hand, z.B. im Windows-Excplorer
 anlegt und die Datenströme als Dateien in dieses Verzeichnis kopiert oder direkt dort anlegt.



## Script-gesteuerte Erzeugung von Objektverzeichnissen

Es spricht natürlich nichts dagegen, die Objektverzeichnisse 
mit eigens dafür geschriebenen Programmen zu erzeugen.

Ein solches Beispiel wäre etwa, dass die inhaltlich verantwortlichen 
die Daten in einem Google Sheet verwalten und ein Script daraus die 
Objektverzeichnisse erstellt.
